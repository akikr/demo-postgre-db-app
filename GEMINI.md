# Gemini Project Helper

This file provides context to the Gemini AI assistant to help it understand and work with this project.

## Project Overview

This is a Java Spring Boot application that provides a REST API for managing bookmarks. It uses a PostgreSQL database for persistence. The project is built with Maven and is configured to use Java 21.

## Key Technologies

- **Language:** Java 21
- **Framework:** Spring Boot 3.5.3
- **Build Tool:** Maven
- **Database:** PostgreSQL
- **Local Development:** Docker Compose
- **Testing:** JUnit, Testcontainers

## Common Commands

- **Build the application:**
  ```bash
  ./mvnw clean package
  ```

- **Run the application:**
  ```bash
  ./mvnw spring-boot:run
  ```

- **Run tests:**
  ```bash
  ./mvnw test
  ```
